# NANANG - Professional Portfolio

Modern, premium "Gold & Navy" themed portfolio with multi-language support (English/Indonesian) and Dark/Light mode toggle.

## Features
- **Modern UI**: Built with a sleek Gold & Navy aesthetic.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Multi-language**: Seamless English and Indonesian translations.
- **Dark/Light Mode**: User-persistent theme toggle.
- **Animations**: Smooth scroll animations using AOS.
- **Contact Form**: Ready-to-connect functional contact UI.

## Technologies Used
- HTML5 & Vanilla CSS
- JavaScript (ES6+)
- [Boxicons](https://boxicons.com/)
- [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
- [FlagCDN](https://flagcdn.com/)

## Deployment
This project is ready to be hosted on **GitHub Pages**, **Netlify**, or **Vercel**.
